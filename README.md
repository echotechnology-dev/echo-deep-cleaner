# Echo Deep Cleaner

Echo Deep Cleaner is a lightweight system utility designed to remove
leftover files, caches, and unused data without running background services.

It gives the user full control over what is cleaned.

## Features
- Removes leftover files and caches
- No background processes
- No telemetry
- Full user control
- Portable build

## Design goals
- Transparency over automation
- No aggressive cleaning
- Predictable behavior

## Status
Beta

## Philosophy
Independent software. Privacy by design.
